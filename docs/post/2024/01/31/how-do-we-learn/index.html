第1章 能力という虚構

  • 人間は事象の原因を探りたがる生物
  • アブダクション（仮説推論） : 原因を推定すること
  • 知覚できない抽象概念はメタファー（比喩）によって理解される
  • 能力
    • 「…力」が仮説として採用される
    • 「力」というメタファーが用いられている
  • 力
    • 内在性 : なにかをなし得るために、その個体に備わっている
    • 差別性 : 程度、強弱
    • 安定性 : いつどこでも一定に出力できるか
  • 能力（というか認知）は文脈依存性をもつので、「力」のメタファーはふさわしくない
    • 内在性・差別性・安定性をもたない
  • 論理的推論や数学的思考力を計測する問題とされているものも、文脈によって正答率が変動するので、実は計測できていない

第2章 知識は構築される

  • 本や他者から伝わって記憶されるのはただの情報
  • （有用性を持つ）知識の性質
    • 一般性 : いろいろな場面で使える
    • 関係性 : 他の知識との関係
    • 場面応答性 : 必要とされる場面で発揮できるか
  • 構成主義
    • 情報は上記3つの要素が揃ってから、主体的に知識として構成される
    • 知識が構成される場面は意識下・無意識下を問わない
      • 本書では「知識の構成」を「知識の創発」と呼ぶ
    • 知識は属人的
      • 新たに構成される知識は、個々人の経験や今までに蓄えた関連知識によって異なってくる
  • 身体性認知
    • なにかを知覚すると、それに対応する感覚が活性化する
    • その感覚の情報（入力・出力）とそれに対する感情が他の感覚の入力となり、連鎖的にあらゆる感覚が活性化する
    • 類似する経験を重ねることで、感覚のネットワークが形成されていく

    • 身体を介したさまざまな感覚経験の記憶が、一般性、関係性、場面応答性を備えた知識として「構築」される

    • 身体化された知識
  • 情報伝達における言葉の欠点
    • 言語隠蔽効果 : 全体性を持つような対象、直感的な理解は、言葉で表現しづらい
    • 無意識下の情報は言葉で表現できない
    • そもそも言葉は多義的
    • -> 認識の齟齬
  • 環境と認知
    • 相互作用
      • 環境（状況）から情報を認知する
      • 身体が生み出す行為を通して環境を変化させる
      • 差異の情報を認知する
    • 環境は知識の中に組み込まれている
    • プログラムのフローチャートみたいに手順のすべてを記憶、丸暗記しようとするのは脳のキャパが足りず逆効果
    • 環境には他者も含まれる
      • 優秀なマネージャーは部下を常に監視したり進捗を記憶するのではなく、部下の顔を見てそいつに頼んだ仕事をその都度思い出して連絡しあう
    • オフロード : 環境という外部記憶を利用して認知的負荷を軽減する
  • 知識
    • ある目標を達成するため、それに誘発された認知的リソース（記憶）と状況リソース（知覚）から生み出されるシミュレーションを通して組織化されるもの

第3章 上達する - 練習による認知的変化

  • ある意識化された運動は、無数の意識できない運動の調整から創発されるものなのだ。

  • 練習のベキ乗則
    • T = N * P^c
      • T : 練習の所要時間
      • N : 課題の難易度
      • P : 練習回数
      • c : 学習率（-1 <= c <= 0）
    • -1の場合（T = N / P）は初心者ほど練習効果が高く、熟練者ほど練習効果が低い
    • 0の場合（T = N）は練習してもまったく効果なし
  • 練習を重ねると
    • マクロ化 : 作業の細かいステップ（動作の確認 -> 動作の実行など）が、1つのステップに集約される
    • 並列化 : 左右の手などで別の動作を同時に行う
    • 結果、動作が無意識下で実行される
  • スキルと実行環境は一体化している
    • スキルを実行しやすいように環境が変えられる
    • 環境はスキルを円滑に実行するための情報を提供する
  • 人間は完全な実行環境がなくても脳内で仮想的に元の環境を再現可能
    • 実際の練習とイメージ・トレーニングを交互に行うと、実際の練習のみを行うよりも効果あり
    • ただし、初心者には効果なし（動作のイメージがまだ定着してないから）
  • ベキ乗則からの逸脱（スランプ）
    • 以下の時期が繰り返し発生し、近似的にはベキ乗則に等しくなる
    • 停滞期 : あまり上達しない
      • 複数の操作方法が拮抗、多様なリソースが存在する冗長な状態
      • 試行錯誤している
    • 後退期 : できていたことができなくなる
      • 新たな操作方法を主に使っている
      • 前後の操作（と実行環境）は以前の操作に最適化されているので、新たな操作向けにはまだ調整されていない
    • ブレークスルー : 急激に上達する
      • 新たな操作方法は以前のものより優れており、それが定着した

第4章 育つ - 発達による認知的変化

  • 発達
    • 加齢による非可逆的変化
    • 上達と違い、特別な練習は必要ない
  • 発達段階論
    • 子供の発達は段階的（デジタル的）であるとする考え
      • 実験により、そうでもないことがわかってきた
    • 段階によって世界観がまったく違う
    • 「子供は小さい大人」という旧来の価値観を否定
    • 教育に児童中心主義をもたらした
  • 上達と同じく、発達も変化はアナログ的だし揺らぎがあったりする
  • 経験からのフィードバックにより複数の認知リソース（考え方）が利用可能になる
    • 段階間の質的変化と呼ばれていたものは、各認知リソースの利用頻度の変化のこと
