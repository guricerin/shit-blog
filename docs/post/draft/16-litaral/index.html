活字

2021-11-16

映話回線では、いろいろな番組が流されている。視聴者参加のバラエティから、ソープオペラ、宗教番組などなど。もちろんニュース専門局もある。おれはニュース以外はほとんど観ない。楽しみは、本棚に並んでいる活字から得る。本はうるさくなくていい。

神林長平『ライトジーンの遺産』P.308 ハヤカワ文庫

件の小説は、おれの精神が不安定なときにたびたび読み返しているものだ。「映話回線」という古めかしい表現が示すとおりジャンルはSFだが、それに目をつぶればもうほとんど現実世界の描写と区別がつかない時代になっている。

読書が好きだったこともあり上記の文章を初めて読んだときは感銘したものだが、読み返してみて思うところがある。最近は本ですらうるさい。活字すら目障りになり得るということがわかってきた、と言ったほうが正しいか。

CSSや文字修飾など見た目だけの問題ではない。言葉の表現の仕方によっては、すさまじいまでのノイズになる。それこそ、心身に支障をきたすレベルに。

辺りを見渡せばゴミ情報が漂っている。いや、ゴミ情報ならまだいいほうだ。誰かを中傷しているなどというわけではないにも関わらず、中には心の底からの嫌悪感を掻き立てる言葉がある。「嫌悪感を掻き立てること」そのものを目的とした芸術作品なら話は別だが、たいていの場合そうではないだろう。

「そんなもの見なければいい」と言うだけなら簡単だ。そうではないから困っている。趣味のため、社会生活を営むためといった、何かしらの目的で情報を集めるときに必ず目に入ってしまう。自分にとって有用か有害かを選別する際に、無意識下であろうが脳は必ずすべての言葉を認識している。ノイズはその脆弱性を突いてくる。

これは別に視界に限った話ではなく、他の五感についても同様だろう。ただひとつ、視界とそれ以外の感覚との違いは、一度摂取してしまった毒を排出する術をもたないこと。それは聴覚にしても似たようなものだが、毒の「反響」は明らかに視覚のほうが強い。映像は脳内スクリーンにこびりつき、やすやすと忘れさせてはくれない。そういった身体機能を利用されているような感じが、たまらなく不快だ。

そういう情報表現を行う者は自らの生存を脅かす敵だ。と言ってしまうのは、大げさだろうか。
