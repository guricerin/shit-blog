brainfuck方言のB'z言語を作った

2020-11-24
プログラミング, B'z

レビューさぼって何書いてんだおれは。

B’z言語の命令とbrainfuckの命令の対応は次の通り。

bf B’z言語
> ウルトラソウッ！
< ハァイ！
+ B’zの、
- シンパイナイモンダイナイナイナイザッツライフイッツオーライ
. B’zのLIVE-GYMにようこそ！！！
, 松本に相談しようか
[ （´.-`） |
] （■∋■）ヴェイ！

以下のbrainfuckコードがあったとする。

>+++++++++
[<++++++++>-]
<.>+++++++
[<++++>-]
<+.+++++++..+++.
[-]
>++++++++
[<++++>-]
<.>+++++++++++
[<+++++>-]
<.>++++++++
[<+++>-]
<.+++.------.--------.
[-]
>++++++++
[<++++>-]
<+.
[-]
++++++++++.

B’z言語の文法規則ファイルを用意する。

{
    "rshift": "ウルトラソウッ！",
    "lshift": "ハァイ！",
    "inc": "B'zの、",
    "dec": "シンパイナイモンダイナイナイナイザッツライフイッツオーライ",
    "write": "B'zのLIVE-GYMにようこそ！！！",
    "read": "松本に相談しようか",
    "loop_begin": "（´.-`）",
    "loop_end": "（■∋■）ヴェイ！"
}

これを使って、前回紹介した拙作ツールでbrainfuckコードをB’z言語に変換する。

$ bft /path/to/code.bf -t /path/to/bz-lang-grammar-json > code.bz
