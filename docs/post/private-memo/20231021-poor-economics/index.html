『貧乏人の経済学』個人メモ

2023-10-21
個人メモ

第1章 もう一度考え直そう、もう一度

  • 貧困層: 1日1人99セント（約120円）以下で暮らす人々
  • 貧困国に対する援助について
    • 賛成派: リソースを与えりゃなんとかなる
    • 否定派: 被援助国の政府を腐敗させたり、貧困層の自律性を奪う
    • どっちも大雑把な推測に過ぎない
    • リソースが最終的にどこにたどり着くか、どう使われるのかを見てない
  • 貧困の罠にまつわるS字曲線 p.29
    • 貧困の罠ゾーン
      • 将来得られるリソースは今日得られるものよりも少ない
      • どんどん貧乏になる
    • 貧困の罠の外側ゾーン
      • 将来得られるリソースは今日得られるものよりも多い
      • どんどん豊かになる
      • ある程度まで豊かになったら増え方が緩やかになる
  • 著者たちはランダム化対照試行を使って、様々な条件下の貧困層の行動を分析

第2章 10億人が飢えている？

  • 大規模な食糧援助は、物流的に悪夢
    • 途中でねずみに食われたり、（書いてないけど）腐ったり盗まれたりで、大半を失う
  • 食料が豊富な国にも餓死寸前の貧困層が存在する
    • 臨時収入が入ったら嗜好品や高級食材に使っちゃう
    • 高級食材と言っても、雑穀よりは美味しい米や小麦、砂糖、エビ、肉とかだけど
    • 結果、カロリー摂取量は援助前と変わらない（むしろ減ったかも）
  • インドではあらゆる階層の家計における食費比率が減少している
    • 中身も、より美味しく高級な食材に使われる
  • 子宮にいた時期や幼少期に栄養状態の良かった成人は、平均すると身長が高く頭も良い
    • また、IQの高さが収入の高さにつながる
    • 身長が高いと収入が良い、というのは偽相関
  • 単にカロリー摂取量を増やすだけでは、生産性向上への影響は小さい
    • 成人には貧血症予防の鉄分が、生産性向上にでかい影響を与える
  • 貧困層は値段が安かったり栄養価の高い食料より、美味な食料を選ぶ
    • てか先進国だって栄養の良い食事についてたいして分かってない
  • 貧困層は結婚式や葬式などの儀式にカネを使いがち
    • グループにおけるメンツの問題
  • テレビやケータイなど娯楽にもカネを使う
  • 人間は退屈には勝てない
  • 貧困層は投資や機会に懐疑的
  • 子供や親に栄養補給剤を摂取したくなるインセンティブを与えることが効果的
    • 人々が食べたがる食物に栄養素の粉末を混ぜたり、美味くて健康的な食物を作ったり

第3章 お手軽に（世界の）健康を増進？

  • 子供の死因として脅威な下痢を予防するのは安上がりなのに、たいして活用されない
  • 安い予防医療ではなく、病気になってから高額な医療費を払うはめになる
  • 公共の医療機関より、高価で技術もお粗末な民間の医療機関に頼りがち
    • 公共の医療機関の無断欠勤率が高い
    • 心理的埋没費用（人々は価格で品質を判断しがち）のせい？違う。
    • 正しい情報や知識が不足してるから。先進国だってデマに踊らされて予防接種を受けない場合がある。
  • 人々はコストを後回しにしがち
    • 予防接種に並ぶ時間がもったいない
    • 安価で高価のある薬を買うよりも、美味いものを買う
    • ジム通いをさぼる
  • 希望するものが予防接種などを受けるのではなく、予防接種などを受けるのがデフォルトとして希望しない者を不参加にさせるという方式が良い
    • 少しの意思決定でさえもコストになるから
    • 行動経済学のナッジで、善い選択肢へと誘導させる
  • 人間の自制心と決断力は限られている
    • 環境に恵まれるほど、それらに依存する必要はなくなる
    • 先進国はナッジに囲まれているので、意思決定のリスクが貧困国と比べてほぼ皆無

第4章 クラスで一番

  • 貧困国で就学率は上昇しているが、教師がサボっている
  • 学習結果も散々なもの
  • 教育効果と将来の収入
    • 初等教育が最も高い。教育年数が増えるごとに効果はなだらかになる。
      • ついでに言えば、初等教育を受けてない大人にそれを受けさせると収入が増えやすい
    • 「高等教育が最も高い」と誤解されがち
    • なので、最も頭のいい子供だけに教育資金を注ぎ込み他の兄弟は働かせる、みたいなことになりがち
    • 教師もこうした誤解をしているので、学校はエリート主義的になっている。成績の低い生徒に対するやる気がない。
    • なんなら、高カーストの教師よりも、低カーストの教師のほうが、同じ低カーストの生徒に対する信頼感がない
    • 子ども自身もこうしたステレオタイプを内面化しちゃってる
  • 才能を確認する正攻法: 多くの時間をかけて、何が得意かを本人が示せるだけの機会を与えること
  • 成績の良い児童には自分のペースで学習させ、悪い児童に重点的に基礎を教えると、結果としてすべての児童の成績が伸びる

第5章 スダルノさんの大家族

  • 子供を減らしても、その分だけ親から子供への投資を集中できるわけではない
  • 貧乏で教育を受けていない女性は、そうでない女性に比べて避妊をしない
  • 10代女性からすると、金持ちおじさんの子供を産むことは、その人が自分の面倒をみざるを得なくなると考えるなら、魅力的な選択肢
    • とくに学校に通っていない女性の場合、実の家族から疎まれるよりも自分の家庭をもつほうがよい
    • ただし、高年齢男性とのセックスのほうがHIV罹患率が高いと知ると、セックスを避けるようになる
  • 夫婦が対立しており夫のほうが権力が上の場合、避妊薬を渡す方法を工夫する必要あり
    • 妻にこっそり渡すほうが、避妊率は高くなる
    • 男性は女性に比べて大家族志向。女性は肉体的負担から、妊娠を避けたい。
  • 避妊率は社会規範・宗教規範も影響する
    • 子供を増やさない夫婦が冷遇されるようなら、避妊率は低いまま
  • 先進国の老後の保障
    • 社会保障、投資信託、健康保険など
  • 途上国の老後の保障
    • 子供が自分の面倒をみてくれるかどうか
  • 将来子供からもらえる期待金額が低いと考えるなら、子供じゃなく金融資産に投資される
  • 胎児が女性だと中絶される確率が上がる
    • 娘の結婚式には持参金が必要
    • 結婚すれば経済的には夫に支配されがち
    • 親の老後の面倒をみるのはたいてい息子
    • というわけで、娘を産んだとしても投資しないしネグレクトもしうる
    • -> 世界で女性が男性より若干少ない

第6章 はだしのファンドマネージャ

  • 貧乏人の大半は、小事業者・農家・日雇い労働者
    • 利益は出るのか、そもそも仕事はあるのか、不確定な状況
    • 変動幅がでかすぎる
  • 貧乏人のリスク
    • 健康、政治的暴力、犯罪被害、汚職
  • リスクまみれな生活の貧乏人にとっては、世界規模の経済危機は鼻くそレベル
    • そういうのはむしろ中間層以上の者にダメージを与える
  • 貧乏（所得リスク）でいるだけで、健康リスク・心理的リスクが跳ね上がる
    • ストレスによって知覚力・意思決定力が低下し、経済的に合理的な選択をしずらくなる
  • 貧乏人のリスク対応手法
    • 従事する職業を分散する -> 専門家になれない
    • 大家族や宗教・民族コミュニティ内で助け合う、非公式の保険・融資を行う
      • -> 健康上の打撃の場合だと機能しない。食物を分け与える程度なら貧乏人でも可能だが、効果な医療費だと話が違ってくる。
    • 保守的な農業を行う -> 革新的な手法はわずかにリスクはあるが、ずっと利益率が高い
    • 小作農になる -> インセンティブが低い
    • たいてい効率が悪い
  • 貧乏人が保険商品を買いたがらない理由
    • 保険の概念を理解できないからではない
    • 保険会社を信用できないため
    • 将来のリスクについて検討するのが難しいため、ていうか最悪のシナリオはできるだけ考えたくない（これは貧乏人に限らない）

第7章 カブールから来た男とインドの宦官たち - 貧乏人融資のやさしい（わけではない）経済学

  • 高利貸しに借りるしかない貧乏人が、マイクロファイナンスが登場により貧困を脱却
  • それでもまだまだ貧乏人がいるのはなぜか
  • どの国どの銀行でも貧乏人ほど高金利の融資となる
    • 貧乏人ほど債務不履行率が高いからではない
    • 貸し手側には、借り手が逃げないよう信用調査・監視を高頻度で行う必要があり、これがコストとなって高金利に反映されるため
    • 公式の金融機関は割に合わないため貧乏人には貸さず、闇金などが脅しでもって貸し付ける
  • マイクロファイナンスの効率の良い融資管理
    • グループに対して融資を行う。返せなかったら連帯責任。
    • 返済を行う場所・時間は週ごと・グループごとに決まっている
    • 取引がシンプルなので融資担当者の教育も不要
    • ルールが多く柔軟性がないので、高利貸しからマイクロファイナンスに乗り換える率は思ったより高くない
  • マイクロファイナンスは起業向けではない
    • 貧乏人に起業する能力はない
    • そもそもマイクロファイナンス側としては借り手にリスクを取らせず、安全な経済行動をとってもらいたい

第8章 レンガひとつずつ貯蓄

  • 少額貯金は引き出し手数料が割高なので、実質的に貧乏人には銀行を利用できない
  • 貧困層グループでの非公式な貯蓄方法などはあるが、複雑すぎる
  • 貯金口座をもつ貧乏人は持たない貧乏人に比べて、貯金額や投資が多く、病気になったときの運転資金食いつぶし率が少ない
  • でも色んな誘惑やらのせいで貯蓄というのは難しい
  • 人間は「明日の自分は今日の自分よりも堪え性があるはずだ」と願いがち
  • とはいえ誘惑財をほいほい購入して貯蓄できないと、一生貧乏のまま
  • 高ストレス下で生きていると衝動的な意思決定をしてしまう
  • 社会保障や貯蓄口座などのセーフティネットがない貧乏人にとっては毎日がストレス
  • 逆に、ちょっとした希望と励ましや落ち着きがあれば、貯蓄へと誘導できる
  • はじめは小さくていいので複数の目標を達成させていく。そうして成功体験を積ませ、自信をつけさせる。

第9章 起業家たちは気乗り薄

  • たいていの貧乏人の事業
    • 従業員0
    • 資産ほとんどなし（仕事部屋すらめずらしい部類）
    • 継続性なし（数年後には営業終了）
    • 限界収益は高いが、総収益は低い。しかも限界収益が高いのは最初のほうだけ。
  • 貧乏でも才能ある人間の事業成功例というのはレアケース
  • 総収益と限界収益 P.283
    • 総収益 : 総売上から事業経費（原材料費や賃金など）を差し引いたもの
    • 限界収益 : 投資を増やしたとき、投資先事業の総収益がどれだけ変化するか
  • そもそも起業・事業運営は貧乏人じゃなくても難しい
  • 貧乏人が貯蓄しない理由 : 可能な範囲の貯蓄をしても欲しい消費目標にたどり着けないため
  • 貧乏人が事業に投資しない理由 : 可能な範囲の投資をしても結果は大差ないため、手間が増えるだけのため
  • そもそも貧乏人が事業を営む理由 : 定職に就くには応募資格、経験、時の運が必要なため
  • 貧乏人は子供には公務員になってほしい
    • 雇用が安定している
  • 公務員じゃなくても安定した雇用があると人々は一変する
    • 数世代かかるかもしれないが、中流階級に上がれる可能性が増大する
    • 「未来のために働く」という意識が芽生える
  • 人間が長期的な見方をできるようになるには、安定性の感覚が必要
  • よい仕事（安定しており高給）のほとんどは都市部にある
    • ただし都市部に永続的に引っ越せる人間は少ない
    • 物価が高いため
  • 知り合いが既に都市部にいる場合は引っ越し率も高くなる
    • 職探し、急病人が出た場合に助けてくれるなどセーフティネットとなっている
  • 都市部に低所得者向けの住宅やインフラを整備するのは重要だが、地方のインフラを整備するのはさらに重要
    • 地方でも「よい仕事」を創出できるから
  • マイクロ融資は貧乏人の当分の生活を保障するものの、貧困からの脱出にはなりえない

第10章 政策と政治

  • 援助は汚職によって中抜きされやすい
  • 汚職の防止には、行為者に対する適切な監視だけで十分で、政府の転覆や社会の根本的な変革は不要かもしれない
  • 植民地時代に作られた、支配者に都合の良い政治制度・経済制度は、現在にも引き継がれている
    • 現在の権力者に都合が良いから
  • 悪い制度を外国の手で排除し民衆に政治的・経済的自由を与える戦略は危険だし、民衆が自力で悪い制度を排除し自由を勝ち取るという楽観論も駄目
  • 規模の小さい制度（町村単位など）の変化やちょっとした技術変化が、驚異的な効果をもたらす場合がある
    • 投票用紙による選挙（字を読めない人間は排除される）から、単純な電子投票（画面に候補者の姿を映す）への変化により、貧乏人の有効票が増えるなど
    • 中抜きする地方役人に対して中央役人が少し脅すだけで、汚職率が減ったり
  • 能力ではなく、その地域で優勢な民族出身という理由で選挙に勝った候補者の場合、汚職率が跳ね上がる
    • とはいえ有権者は民族やカーストで選んでるわけではない
    • 候補者についてほとんどなにも知らない
    • だったら少しでも共通点の多い、同じ民族・カーストの候補者を選ぶ、という傾向にあるだけ
  • 悪い政治制度下でも良い政策は生まれ得る。逆に、良い政治制度下でも悪い政策は生まれ得る。
    • 現場の労働環境を何も知らない、善意の政治家・官僚の政策は失敗する
    • 高望みで難易度の高い要望は叶えられず、単純な作業や目標であるほど達成されやすいし、その成果も分析しやすい
  • 政府が実績を出せば民衆は政治を真面目に考えたり、政府を信頼したり、さらなる成果を政府に期待するようになる
    • 選挙の棄権、やけくそな投票行動、テロリズムも減る

網羅的な結論にかえて

  • 教訓
    • 貧乏人は重要な情報を持っていないことが多く、まちがったことを信じている
      • 正しい情報を知らせるには、信頼できる情報源が、魅力的で単純な方法を使う
    • 貧乏人にはセーフティーネットがないため重大な意思決定を頻繁に行う必要があり、これが高ストレスとなる
      • 金持ちであるほど、判断を他人にやらせることができる
      • 人間の先送りグセが邪魔をしてくる
    • 貧乏人には一部の市場が提供されておらず、あっても不利な価格に直面する
      • これを改善するには政府の介入が必要
    • 貧困国が失敗続きなのは、エリート層の陰謀によるものではなく、政策設計の単純なミスであることが多い
    • 人々に何ができて何ができないかという期待は、良くも悪くも予言の自己成就となる
